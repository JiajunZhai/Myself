# Hello World! 🚀

欢迎来到我的数字花园！这里由 **Astro 4.x** + **Tailwind CSS** + **MDX** 驱动，是一个默认零 JavaScript 加载、性能极佳的静态站点。

## 为什么选择 Astro？

> Astro 完美契合了“内容为王”的理念，而且“孤岛架构”让它拥有不可思议的加载速度。

- **极速**：默认零 JS
- **灵活**：想用 Vue、React 随时可接入
- **内容聚合**：Content Collections 写起 Markdown 来简直不要太爽。

## 代码支持演示

```js
function greet(name) {
  return `Hello, ${name}! Welcome to the cyberspace.`;
}
```
