# The Digital Garden Manifesto 🌌

For years, I've been scattering my thoughts across platforms I don't truly own. Today, that changes. This space is not just a "blog"—it's a **Digital Garden**.

A digital garden is a living, breathing cyber-organism. It's an intersection between a notebook, a public portfolio, and a chaotic playground. Here, ideas are allowed to be born as rough drafts, nurtured over time, and eventually blossom into comprehensive concepts.

## 🏗️ Architecture

I built this sanctuary utilizing a hyper-optimized tech stack:

- **Astro v6**: Providing a robust static-site generation engine that ships exactly 0kb of JavaScript by default.
- **Tailwind v4**: The ultimate utility-first styling engine, rendering glassmorphism and 3D tilts natively.
- **Keystatic**: A local-first CMS that lets me write `.mdx` directly visually, without the bloat of a database.

### The Physics of the UI

If you hover over the cards on the homepage, you'll notice a tactile, 3D tilt response. This isn't just for show. It represents a **frictionless connection** between the static content and the physical user interacting with it. It's a design philosophy that respects the visitor.

```javascript
// The gravity formula behind the 3D cards
VanillaTilt.init(document.querySelectorAll(".card"), {
  max: 25,
  speed: 400,
  glare: true,
  "max-glare": 0.5,
});
```
